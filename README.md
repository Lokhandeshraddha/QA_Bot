# LLM-Based Question Answering Bot

An AI-powered Question Answering system built using Hugging Face Transformers.  
The application uses a pre-trained LLM to answer user queries based on a given context, demonstrating practical LLM integration for real-world use cases.

Features
- LLM-powered Question Answering
- Uses Hugging Face `pipeline` API
- Custom context-based responses
- Ready for UI integration using Gradio

 Tech Stack
- Python
- Hugging Face Transformers
- PyTorch
- Gradio (for UI)
